# imis

R client for IMIS

## Installation

Installing `IMIS` requires the `devtools` package:

```R
install.packages("devtools")
devtools::install_github("Daphnisd/hlptools") 
devtools::install_github("EMODnet/imis")
```


## Usage

```R
flatdatasetrecord (4447)
flatdatasetrecord (c("4447","5444"))
```
